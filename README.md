# Calories Log - Your Healthy Web App

**Calories Log** is a sleek and intuitive web app built with **React.js** and **Chakra UI** to help you track your daily calorie intake effortlessly. It features **BMI calculation**, a collection of **healthy recipes with calorie counts**, and **the latest health news** to keep you informed.
